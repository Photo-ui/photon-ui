To make the label float inline add the floatingLabel attribute and set the value to inline this way floatingLabel="inline" .


pi-text-area label="Address" floatingLabel="inline"/pi-text-area

To make the label float outline add the floatingLabel attribute and set the value to outline this way floatingLabel="outline" .


pi-text-area label="Address" floatingLabel="outline"/pi-text-area

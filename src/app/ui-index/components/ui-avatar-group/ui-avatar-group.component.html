Avatar Group

With group avatar we provide a list of images to the [images] property which is displayed in a stacked order. 


                            {{avGrp}}
                        
                            {{avGrpComponent}}

With group avatar we can provide a number to show the number of images we wish to display using the [maxCount] property. If the image length is greater than the [maxCount] property number, the remaining image count is displayed at the end of the stack. 


                            {{avGrpMaxCount}}

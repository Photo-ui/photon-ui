Avatar


The avatar component supports four different sizes small normal large
extra large and it takes a property image 


                            {{avSizes}}
                        
                            {{avComponent}}

This is an avatar with the rounded property. This property ensures the borders of the avatar are fully rounded. 


                            {{avRounded}}

This is an avatar with the rounded="none" property. This property ensures the borders of the avatar are not rounded. 


                            {{avRoundedNone}}

This is an avatar with the rounded property not provided. This property ensures the borders of the avatar are only rounded at the edges, this is the default property of the avatar component. 


                            {{avNoRounded}}


With this avatar we do not provide an image instead we provide the initials property. 


                            {{avInitials}}

This is the default state of an avatar when neither the initials nor image property is provided. 


                            {{avEmpty}}

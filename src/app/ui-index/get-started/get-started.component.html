Get Started

TolleUI is a set of angular UI components.


Download

TolleUI is available on npm and you can add it to your existing application by running the following command.
TolleUI uses primeicons as it's icon library so you would have to install it too.
* This project is still in it's early stages.

npm install toll-ui
npm install primeicons --save

Import

The components are configured as modules and each component is imported from "toll-ui".
Below is how an import would look like

import { PiButtonModule } from "toll-ui"

Styles

You have to apply two styling dependencies one for tolleUI and the other for primeicons angular.json

"styles": [
    "src/styles.css",
    "node_modules/toll-ui/core/style.scss",
    "node_modules/primeicons/primeicons.css"
],
